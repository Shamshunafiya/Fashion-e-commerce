# Zaro Fashion — Contact Page

A pixel-faithful React + Tailwind CSS recreation of the Zaro Fashion eCommerce contact page.

## Tech Stack
- React 18
- Vite
- Tailwind CSS 3
- Google Fonts: Cormorant Garamond + Jost

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
